This dataset contains results for the VERA 3 tests for the academic years 2009/10-2018/19 and is licensed under CC-BY 4.0 https://creativecommons.org/licenses/by/4.0/.

Description of variables:
id: 	    Federal state identifier
jahr:       Calender year in which test took place (i.e., 2010 refers to the academic year 2009/2010)
subject:    Test subject (German or Mathematics)
test:       Test domain within subject. Note that for Hamburg only the global proficiency level for mathematics is available. 
bundesland: Federal state name 
k1_anteil - k5_anteil:  Share of students (in %) scoring at a specific proficiency level (1=lowest,5=highest). For more information on the proficiency levels, see: https://www.iqb.hu-berlin.de/bista/ksm

Source:
